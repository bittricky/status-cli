# status-cli

> Check if a website is up or down

## Install

Ensure you have Node.js version 8+ installed. Then run the following:
```
$ npm install --global status-cli
```

## Usage

```
$ status -h

  Usage
    $ status -f url_list.txt
    $ status -u http://example.com
```
